# How to Build

여기서는 HANU Community의 주요 Project를 Build하기 위해 공통적으로 요구되는 사항을 제공합니다. 

## Docker로 빌드하기

HANU는 보다 쉽고 빠른 빌드 환경을 제공하기 위해 [Docker](https://www.docker.com/)를 사용합니다. Docker를 이용한 빌드 절차는 다음과 같습니다.
